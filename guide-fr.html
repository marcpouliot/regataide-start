Guide d’utilisation — Regataide Start2

V1 avec GPS du téléphone

Retour app English

But de l’application

Regataide Start2 aide le régatier à gérer son approche de la ligne de départ avec le GPS du téléphone. Ce n’est pas un outil officiel de jugement.

L’app calcule la distance, le côté de la ligne, le temps restant et la vitesse idéale pour atteindre la ligne à la prochaine minute pile.

Convention de la ligne

  • A = bateau comité.
  • B = bouée de départ.
  • Bouée à bâbord = au passage, le bateau laisse la bouée du côté bâbord.
  • Bouée à tribord = au passage, le bateau laisse la bouée du côté tribord.

Coordonnées par défaut : comité A = 45.840722, -71.112139 ; bouée B = 45.9404030, -71.1172860.

Exemple d’utilisation

  1. Ouvrir l’application sur téléphone en HTTPS.
  2. Appuyer sur Activer GPS.
  3. Vérifier ou modifier les coordonnées A/B.
  4. Choisir Bouée à bâbord ou Bouée à tribord.
  5. Appuyer sur SYNC 5:00 au signal 5 minutes, ou entrer l’heure officielle.
  6. Lire la distance, la vitesse actuelle, la vitesse idéale et l’état.

Mode distance

Perpendiculaire : mesure le gap jusqu’à la ligne A-B. C’est le mode recommandé pour le départ.

Centre : mesure la distance jusqu’au milieu de la ligne. Utile pour comparer et déboguer.

Carte

La carte affiche le bateau, le bateau comité A, la bouée B, la ligne de départ, le centre de ligne et le point GAP.

Le bouton Centrer recentre la carte sur le bateau, ou sur la ligne si le GPS n’est pas encore actif.

Configuration avec fichier JSON

Entre un nom dans Nom du fichier, puis utilise Exporter fichier. Le navigateur crée un fichier JSON dans les téléchargements.

Pour reprendre une configuration, utilise Charger fichier et sélectionne le JSON exporté.

Sauvegarder local conserve la configuration seulement sur ce téléphone/navigateur.

Conseils de test

  • Tester d’abord à pied ou en voiture lente dans un endroit ouvert.
  • Utiliser une zone tampon de 10 à 15 m avec le GPS du téléphone.
  • Surveiller la précision GPS affichée. Si elle est ±20 m, les résultats seront approximatifs.
  • En milles/heure, vérifier plus facilement les calculs terrestres. En régate, utiliser les nœuds.
