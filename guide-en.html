User guide — Regataide Start2

V1 with phone GPS

Back to app Français

Purpose

Regataide Start2 helps the sailor manage the approach to the starting line using the phone GPS. It is not an official line-judging tool.

The app calculates distance, line side, time remaining, and the ideal speed to reach the line at the next exact minute.

Line convention

  • A = committee boat.
  • B = starting buoy.
  • Buoy to port = when crossing, the boat leaves the buoy on the port side.
  • Buoy to starboard = when crossing, the boat leaves the buoy on the starboard side.

Default coordinates: committee A = 45.840722, -71.112139; buoy B = 45.9404030, -71.1172860.

Example

  1. Open the app on the phone over HTTPS.
  2. Tap Enable GPS.
  3. Check or edit the A/B coordinates.
  4. Choose Buoy to port or Buoy to starboard.
  5. Tap SYNC 5:00 at the five-minute signal, or enter the official start time.
  6. Read distance, current speed, ideal speed, and status.

Distance mode

Perpendicular: measures the gap to line A-B. Recommended for starts.

Center: measures distance to the middle of the line. Useful for comparison/debugging.

Map

The map shows the boat, committee boat A, buoy B, starting line, line center, and GAP point.

Center recenters the map on the boat, or on the line if GPS is not active yet.

JSON configuration file

Enter a name in File name, then use Export file. The browser creates a JSON file in downloads.

To restore a configuration, use Load file and select the exported JSON.

Save local keeps the configuration only on this phone/browser.

Testing tips

  • Test first while walking or driving slowly in an open area.
  • Use a 10–15 m buffer with phone GPS.
  • Watch the displayed GPS accuracy. If it says ±20 m, results will be approximate.
  • Use mph for easy land testing. Use knots for regattas.
